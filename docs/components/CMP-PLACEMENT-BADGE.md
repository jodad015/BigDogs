# CMP-PLACEMENT-BADGE — Placement Badge

## Description
Visual indicator for weekly placement (1st through 4th).

## Props
| Prop      | Type    | Description           |
|-----------|---------|-----------------------|
| placement | number  | 1, 2, 3, or 4         |
| size      | string  | "sm" / "md" / "lg"    |

## Styling
| Placement | Color  | Label |
|-----------|--------|-------|
| 1st       | Gold   | 1st   |
| 2nd       | Silver | 2nd   |
| 3rd       | Bronze | 3rd   |
| 4th       | Gray   | 4th   |

## Used On
SCR-DASHBOARD, SCR-WEEKLY, SCR-LEADERBOARD, SCR-PARTICIPANT
