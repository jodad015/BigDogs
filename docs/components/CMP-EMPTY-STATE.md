# CMP-EMPTY-STATE — Empty State

## Description
Placeholder shown when a screen has no data to display yet.

## Props
| Prop    | Type      | Description                        |
|---------|-----------|------------------------------------|
| icon    | string    | Icon name                          |
| title   | string    | Heading (e.g., "No weigh-ins yet") |
| message | string    | Descriptive text                   |
| action  | object?   | Optional CTA button {label, onClick} |

## Used On
SCR-TREND (no weigh-ins), SCR-LEADERBOARD (no results yet), SCR-DASHBOARD (first time)
