# CMP-SHOWDOWN-BANNER — Showdown Banner

## Description
Prominent banner indicating the current or selected week is a Showdown
Week with doubled placement points.

## Props
| Prop    | Type    | Description                    |
|---------|---------|--------------------------------|
| active  | boolean | Is this week a showdown?       |
| compact | boolean | Smaller version for dashboard  |

## Display
- Full: bold banner with "SHOWDOWN WEEK" + "2x Points" badge
- Compact: small badge/pill for inline use
- Distinct color treatment (purple/gold per DESIGN_PATTERNS)

## Used On
SCR-DASHBOARD, SCR-WEEKLY
