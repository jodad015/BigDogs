# UC-15 — View Participant Detail

## Actor
Participant in the same challenge

## Description
View another participant's challenge progress: trend chart, weekly scores,
and overall stats. Respects privacy settings (GAP-10).

## Display Elements
- Display name + avatar
- Progress bar toward goal
- Weekly score history (list of score cards)
- Trend chart (if weight privacy is transparent)
- Status (active / maintenance)
