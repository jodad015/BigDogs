# DLG-CONFIRM-GOAL — Confirm Goal

## Description
Final confirmation step in onboarding before locking in the challenge goal.
Shows computed values for review.

## Type
Inline review panel (part of SCR-ONBOARDING Step 4), not a modal.

## Elements
- Summary display:
  - Goal method used
  - Target weight
  - Total loss
  - Weekly target (committed pace)
  - Challenge duration
- "This pace is locked for the entire challenge" disclaimer
- Confirm button
- Back button (return to adjust)
