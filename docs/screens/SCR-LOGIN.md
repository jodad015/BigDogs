# SCR-LOGIN — Log In

## Description
Authentication form.

## Route
`/login` (public only — redirects to dashboard if authenticated)

## Elements
- Email input
- Password input
- Log In button
- "Don't have an account? Sign up" link

## Navigation
- Success → SCR-DASHBOARD
- Sign Up link → SCR-SIGNUP
